sinc
===============

Developed by Deon Poncini <dex1337@gmail.com>

Description
-----------
Sinc is a code generator that focuses on generating code for a variety of
languages mostly around data type definitions. This allows the same data
structures to be used across language bindings.

Usage
-----
Create an XML file defining the data types you would like available in the
host of languages, then run sinc.py to generate code.
